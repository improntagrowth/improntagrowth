# Netlify preview

This project can be shared temporarily with Netlify drag-and-drop.

## Build

```bash
pnpm build
```

Next exports the static site to:

```bash
out/
```

## Upload

Open Netlify Drop and drag the `out/` folder:

```text
https://app.netlify.com/drop
```

This is for temporary client review only. Final production hosting can still move to Vercel once credentials, domain, analytics, and Search Console are ready.
